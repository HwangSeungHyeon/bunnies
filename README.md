# Bunnies

## 👨‍🏫 프로젝트 소개

우리가 익히 알고 있는 ‘당근마켓’에 영감을 받아 개발 해보고자 합니다.
모방은 창조의 어머니 라는 말이 있듯이 당근마켓을 모방해 봄으로서, 팀원들의 개인 능력과 역량을 올리는 것을 목표로 하고자 합니다.
